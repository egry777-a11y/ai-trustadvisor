# AI TrustAdvisor (GitHub + Replit Ready)

AI-powered site and payment safety checker.  
Frontend in React + Tailwind (served via Express).  
Backend in Node/Express.

### Run on Replit
1. Import this repo to Replit.
2. Ensure `.replit` file exists with `npm install && npm start`.
3. Click **Run**.
4. Visit the public Replit URL.

### Local Run
```bash
npm install
npm start
```

Then open http://localhost:4000
